## Navigating this Repository

This isn't just a collection of static summaries; it's a **navigable knowledge base**. 

Each project directory is structured as an explorable deep-dive:

- **README.md**: High-level STAR format summary (Situation, Task, Action, Result) providing a strategic overview.
- **lessons-learned.md**: Hard-won insights and "human-in-the-loop" learnings.
- **technical-findings.md**: Architectural "guts," low-level implementation details, and the "why" behind the code.
- **examples/**: Runnable .NET 10 examples demonstrating key principles (e.g., `dotnet run --file Example.cs`).
- **excerpt.md**: Minified STAR summary optimised for CV inclusion.

Feel free to follow the links between documents to explore the "Knowledge Graph" of my engineering journey.
